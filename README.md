Descripción de la aplicación:
BracketCore es una página web que permite la creación y gestión de torneos de videojuegos, tanto presenciales como en línea.

El sistema permitirá:

1. Crear torneos de videojuegos.
2. Definir modalidad (presencial o en línea).
3. Establecer límite de jugadores.
4. Definir recompensa para el ganador.
5. Confirmar asistencia.
6. Generar reportes analíticos sobre participación y popularidad.

El objetivo es facilitar la organización de torneos y analizar el comportamiento de los jugadores y organizadores mediante indicadores estadísticos.

Para ello, se usarán tres tipos de usuario, administrador, usuarios registrados e invitado. 

Descripción de las carpetas (hasta ahora):
public: contiene las imágenes que se usarán en el proyecto.

src: contiene el app con las rutas de todas las páginas y además contiene las subcarpetas:
         components: contiene el navbar y footer que usaremos en el proyecto general.
         pages: están todas las páginas que contiene el proyecto.
